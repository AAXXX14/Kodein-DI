inline fun <reified T : Any> instance(tag: Any? = null): Lazy<T> (source)

Gets a lazy instance of T for the given tag from a factory with an A argument.

The returned property should not be accessed before calling KodeinInjectedBase.inject.



Parameters

T - The type of object to retrieve.

tag - The bound tag, if any.

Exceptions

KodeinInjector.UninjectedException - When accessing the property, if it was accessed before calling KodeinInjectedBase.inject.

Return
A lazy property that yields a T.
