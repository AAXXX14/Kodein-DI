KodeinWrappedType

class KodeinWrappedType : Type (source)

Wraps a ParameterizedType and implements hashCode / equals. This is because some JVM implementation (such as Android 4.4 and earlier) does NOT implement hashcode / equals for ParameterizedType (I know...).
