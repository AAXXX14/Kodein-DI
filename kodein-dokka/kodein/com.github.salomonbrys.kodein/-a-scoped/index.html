AScoped

abstract class AScoped<in A, out C, out T : Any> : Factory<A, T> (source)

A factory to bind a type and tag into a Scope or an AutoScope.

Parameters

A - The type of argument that is needed to get a C context.

C - The type of context that will be used to get a ScopeRegistry.

T - The singleton type.

_creator - A function that creates the singleton object. Will be called only if the singleton does not already exist in the scope.
