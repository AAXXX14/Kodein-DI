LazyKodeinAwareBase

interface LazyKodeinAwareBase (source)

Base LazyKodeinAware interface.

It is separate from LazyKodeinAware because LazyKodein implements itself LazyKodeinAwareBase but not LazyKodeinAware. This is because there are some extension functions to LazyKodeinAware that would not make sense applied to the LazyKodein object. For example, LazyKodeinAware.withClass, if applied to LazyKodein, would create a very un-expected result.
