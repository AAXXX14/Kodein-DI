KodeinInjectedBase

interface KodeinInjectedBase (source)

Base KodeinInjected interface.

It is separate from KodeinInjected because KodeinInjector implements itself KodeinInjectedBase but not KodeinInjected. This is because there are some extension functions to KodeinInjected that would not make sense applied to the KodeinInjector object. For example, KodeinInjected.withClass, if applied to KodeinInjector, would create a very un-expected result.
