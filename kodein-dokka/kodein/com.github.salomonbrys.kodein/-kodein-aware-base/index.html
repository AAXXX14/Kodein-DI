KodeinAwareBase

interface KodeinAwareBase (source)

Base KodeinAware interface.

It is separate from KodeinAware because Kodein implements itself KodeinAwareBase but not KodeinAware. This is because there are some extension functions to KodeinAware that would not make sense applied to the Kodein object. For example, KodeinAware.withClass, if applied to Kodein, would create a very un-expected result.
