# &lt;init&gt;

`KodeinFragment()`

A base class that manages a [FragmentInjector](../-fragment-injector/index.md) for easy bootstrapping of Kodein.
Injections will be available after `super.onCreate` and will be destroyed after `super.onDestroy`.
