# &lt;init&gt;

`KodeinActivity()`

A base class that manages an [ActivityInjector](../-activity-injector/index.md) for easy bootstrapping of Kodein.
Injections will be available after `super.onCreate` and will be destroyed after `super.onDestroy`.
